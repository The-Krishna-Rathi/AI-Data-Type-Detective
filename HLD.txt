User Upload (CSV/Excel/JSON)
        ↓
API / UI (FastAPI or Streamlit)
        ↓
Profiler Agent (local code)
  - collects samples & stats
        ↓
LLM Schema Identifier Agent (LangChain/OpenAI)
  - receives samples+stats, returns semantic type predictions
        ↓
Conversion Planner Agent (LLM + rules)
  - produces safe transformation plan (code or recipe)
        ↓
Executor Agent (Python Worker)
  - executes Pandas transforms (in sandboxed environment)
        ↓
Validator Agent (local code + LLM check)
  - checks conversion accuracy / data loss
        ↓
Output: cleaned file + JSON report + logs
        ↓
Optional: Feedback loop to update RAG store (examples)
